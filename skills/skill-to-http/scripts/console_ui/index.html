🔑 需要 API Key

控制台已开启鉴权保护，首次访问请填入 API Key。

如何获取？
  1. 查看配置文件 config.json 中的 api_key 字段
  2. 或在服务器执行：
    cat <workspace>/.http/secrets/api-keys/skill-to-http.key
  3. 也可以直接询问部署这个服务的同学 / Agent

Key 只保存在你浏览器本地（localStorage），不会上传到其他地方。

⚡ 极速模式

开启后 Skill 执行速度预计提升 2-4x（30-140s → 8-20s）

原理：为 skill-to-http 创建专用轻量 Agent（stt-runner），减少每次执行注入的 system prompt（155k → ~35k tokens）。

⚠️ 副作用：会在 OpenClaw 中注册一个名为 stt-runner 的 Agent，Gateway 会短暂重启（约 2-5s）

关闭极速模式

将从 OpenClaw 移除 stt-runner Agent，Gateway 会短暂重启。
